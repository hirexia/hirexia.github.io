For a fun learning experience

Hirexia's Hall started when I was doing my 3rd assignment of Web-Based Programming, mostly due to boredom and I just wanted to do something else. I had just recently found that site called Openbooks over on Neocities. So I started copy-pasting the source code of their front page to my github.io, apparently it was encouraged by Openbook's developer, learning HTML by copying block-per-block into our page. That way, we could learn what each block does, why it works that way... or why it doesn't work THAT way, silly stuff really, but fun nonetheless.

Site is still work-in-progress these last two days, I've moved the CSS onto a separate file for this page now. So its not messy like the front page anymore, haha (well, mostly). I figured I'd want to use this site for something, but I'm just not sure what would that be... yet.

Oh well, we'll figure something out. Right? I mean, I kinda always does so... I hope that's the case once again.
